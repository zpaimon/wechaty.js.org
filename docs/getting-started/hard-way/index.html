Requirements#

  1. A Linux/macOS/Windows computer
  2. Internet connection

1. Setup development environment#

  • Linux
  • macOS
  • Windows
apt-get install make
brew install make
choco install make
Node.js installation docs
  • Windows
  • Linux(Debian/Ubuntu)
  • macOS

Instal Node.js for other platforms can be found at https://nodejs.org/en/download/package-manager/

Instal Build Tools#

  • Linux
  • macOS
  • Windows
sudo apt install build-essential
brew install make
choco install make

1. Clone wechaty-getting-started Repository#

git clone https://github.com/wechaty/wechaty-getting-started
cd wechaty-getting-started

2. Install Dependencies#

# npm install
make install

3. Run the Bot#

Notice: Wechaty requires Node.js version >= 10

# npm start
# Or use node to run bot directly
# node examples/starter-bot.js
make bot

You are all set!
