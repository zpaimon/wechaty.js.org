What is Wechaty

A chatbot is a software application used to conduct an on-line chat conversation, in lieu of providing direct contact with a live human agent. A chatbot is a type of software that can automate conversations and interact with people through messaging platforms. — Wikipedia

Wechaty is a modern Conversational RPA SDK for Chatbot Makers which can create a bot in a few lines of code.

Wechaty offers out-of-the box support for the common features you'd expect from turning your IM account into chatbot, and can be easily customised and extended by developers to create a chatbot that is tailored to their precise needs.

Wechaty support lots of popular instant messaging platforms:

And you can write Wechaty in almost all popular programming languages:

Features#

  1. Message processing: receive and send messages, support text, image, audio, video, and attatchments.
  2. Room management: create rooms, add/remove room members, manage topics, etc
  3. Contact management: search by name, alias, tags, get profile data and avatar
  4. Friendship management: search and add new friends, accept friendship requests, etc
  5. Intelligent dialogue Management: Just several configuration can get a task-oriented bot.
  6. last but not least: write once, run on all IM platforms!

import { Wechaty } from 'wechaty'
async function main () {
conswt bot = new Wechaty()
bot
.on('scan', (qrcode, status) => console.log(`Scan QR Code to login: ${status}\nhttps://wechaty.js.org/qrcode/${encodeURIComponent(qrcode)}`))
.on('login', user => console.log(`User ${user} logged in`))
.on('message', message => console.log(`Message: ${message}`))
await bot.start()
}
main()
.catch(console.error)
